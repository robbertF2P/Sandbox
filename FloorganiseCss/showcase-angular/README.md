# @floorganise/css — Angular home mock

Platform 2.0 home page mock using **Angular 19** and `@floorganise/css` (Tailwind v4 semantic aliases).

## Run

```bash
npm install
npm start   # http://localhost:5176
```

## What it demonstrates

- Production light shell (`f2p-app-light`)
- Module header with `#module-dropdown-button` (Floor2Plan smoke-test contract)
- Home tile grid (`f2p-tile-grid`, `f2ps-tile`, `f2ps-clickable`)
- Tile navigation via Angular Router to placeholder module pages

Mirrors the Vue `F2pHomeTiles` component in `../showcase/`.
